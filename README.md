## 2. Tekken Cheat Sheet

### Overview

Tekken Cheat Sheet is a cross-platform Flutter application designed for Tekken 8 players to organize and manage character knowledge efficiently. The application allows users to store key moves, punish options and combo routes while providing a clean and responsive user experience on both desktop and mobile platforms.

### Objectives

This project was created to explore practical application development with Flutter and SQLite while focusing on maintainable architecture, local data persistence and polished user experience.

### Technologies

* Flutter
* Dart
* SQLite
* sqflite
* sqflite_common_ffi

### Key Features

* Character-based move management
* Punish tracking and frame data organization
* Combo management with launcher relationships
* Responsive desktop and mobile interface
* SQLite-backed local persistence
* Import/export utilities for database testing and maintenance
* Database inspection tools for debugging and development

### Skills Demonstrated

#### Flutter Development

* Modular widget architecture
* Reusable custom components
* Responsive layouts for multiple platforms
* Platform-specific configuration for desktop support

#### Database Design

* Relational data modeling
* One-to-many relationships
* Foreign key constraints
* Transaction management
* Data import/export workflows

#### Software Engineering Practices

* Separation of concerns between UI and data layers
* Maintainable code structure
* Reusable services and widgets
* Defensive programming and error handling

#### User Experience

* Modern dark-themed interface
* Contextual tooltips and visual feedback
* Interactive controls and animations
* Focus on usability and navigation

### What a Reviewer Can Evaluate

* Flutter and Dart proficiency
* SQLite integration and schema design
* Application architecture and code organization
* UI/UX implementation quality
* Maintainability and scalability considerations
